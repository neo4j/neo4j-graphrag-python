TASK-002: Add filter classification for SEARCH-compatible predicates
Date: 2026-03-13T14:30:00Z
================================================================================

Summary:
Added classify_filter_for_search() and helper _is_filter_search_compatible()
to filters.py. Returns FilterClassification dataclass with is_compatible flag,
optional Cypher WHERE clause, and params. Compatible = simple comparisons
(=, <>, <, <=, >, >=, BETWEEN) joined by AND only. Incompatible = $or, $in,
$nin, $like, $ilike, nested OR logic.

- src/neo4j_graphrag/filters.py (added FilterClassification, classify_filter_for_search, _is_filter_search_compatible, SEARCH_COMPATIBLE_OPERATORS)
- tests/unit/test_filters.py (added 20 tests covering all operator types and edge cases)

Key Decisions:
- Reuses existing get_metadata_filter() for Cypher generation when compatible
- FilterClassification is a dataclass, not a NamedTuple, for clarity
- SEARCH_COMPATIBLE_OPERATORS set defined separately for maintainability
- None/empty filters classified as compatible with no WHERE clause

TASK-003: Add SEARCH clause Cypher query builder in neo4j_queries.py
Date: 2026-03-13T15:00:00Z
================================================================================

Summary:
Added _build_search_clause_vector_query() to neo4j_queries.py. Generates
MATCH (node:Label) SEARCH node IN (VECTOR INDEX indexName FOR $query_vector
[WHERE ...] LIMIT $top_k) SCORE AS score. Accepts optional FilterClassification
to embed WHERE predicates inside SEARCH clause for in-index filtering.

Files Changed:
- src/neo4j_graphrag/neo4j_queries.py (added _build_search_clause_vector_query, imported FilterClassification)
- tests/unit/test_neo4j_queries.py (added 8 tests in TestBuildSearchClauseVectorQuery class)

Key Decisions:
- Uses node alias "node" to match existing query builder conventions
- Backtick-escapes index name and node label for safety
- Uses $query_vector and $top_k params matching existing parameter naming
- Filter params merged into return dict when FilterClassification provided
- Function is private (_build) since it'll be called internally by retrievers
